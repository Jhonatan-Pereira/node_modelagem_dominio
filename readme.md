# Modelagem de domínio

## Comandos
```sh
npm init -y

npm i -D typescript ts-node-dev tsconfig-paths jest ts-jest

npm i -D @types/jest @types/node

npx tsc --init

npm i uuid

npm i -D @types/uuid

npm i -D @faker-js/faker

npm i -D gerador-validador-cpf
```

## Versão da criação do projeto
- Node: v16.20.2
- NPM: 8.19.4

## Versão do curso
- Node: v18.16.0

## Rodar o projeto
```sh
npm run dev
```

## Rodar os testes

npm test
